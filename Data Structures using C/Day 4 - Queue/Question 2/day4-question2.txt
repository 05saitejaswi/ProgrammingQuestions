Write a program to reverse first K elements of a Queue.

Input Format:

Input consists of a N and K followed by N number of queue elements, in which, 
N represents the number of queue elements,
K represents the number pf the queue elements to be reversed.

Output Format:
Output displays the queue elements after the reversal is done.

Sample testcases 

Input 1                     Output 1
5                           Reversed queue:
4                           4 3 2 1 5 
1 2 3 4 5 

Input 2                     Output 2
5                           Reversed queue:
3                           3 2 1 4 5 
1 2 3 4 5


Time Limit : 10 ms
Memory Limit : 256 kb
Code Size : 1024 kb

Note:

The program will not be evaluated if "Submit Code" is not done atleast once.
Extra spaces and new line charactes in the program will also
result in the testcase failing
